# react-webRTC 🚀

바닐라로 구현한 webRTC는 여기로.. [Link](https://github.com/mugglim/nomad-zoom-clone)

## Server Port

-   Client : 3000
-   Server : 4000

## Setup

### 1. install package

```bash
# client
cd /client
npm install

# server
cd /server
npm install
```

### 2. Run Server

```bash
#client
cd /client
npm start

# server
cd /server
npm run dev
```

### 3. Open your browser and go to localhost:3000
